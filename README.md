📖 About the Project


3D Café is a production-grade, full-stack food ordering and restaurant management platform built with the MERN stack — crafted to deliver an immersive café experience through interactive 3D visuals, real-time order tracking, table reservations, and role-based dashboards.



This project was built as a portfolio-grade application demonstrating end-to-end full-stack development: from 3D rendering with React Three Fiber and real-time communication via Socket.io, to secure JWT + OAuth authentication, Stripe payment integration, and containerised deployment with Docker.

The frontend runs fully in Demo Mode — no backend required — making it instantly deployable to Vercel or Netlify for live demonstrations.

⚙️ Installation

⚡ Option 1 — Frontend Only (Demo Mode)

Zero configuration. No backend, no environment variables.

bash# Clone the repository
git clone https://github.com/YOUR_USERNAME/cafe-3d.git

# Navigate to frontend
cd cafe-3d/frontend

# Install dependencies
npm install

# Start development server
npm run dev

✅ Frontend: http://localhost:5173 — use the demo credentials to log in.


🗄️ Option 2 — Full Stack Setup

Step 1 — Configure backend environment

bashcp backend/.env.example backend/.env

envMONGODB_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
JWT_REFRESH_SECRET=your_refresh_secret
GOOGLE_CLIENT_ID=your_client_id
GOOGLE_CLIENT_SECRET=your_client_secret
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
STRIPE_SECRET_KEY=your_stripe_secret
STRIPE_WEBHOOK_SECRET=your_webhook_secret
SMTP_HOST=your_smtp_host
SMTP_USER=your_smtp_user
SMTP_PASS=your_smtp_password

Step 2 — Start the backend

bashcd backend
npm install
npm run seed    # Seed database with demo data
npm run dev

✅ Backend: http://localhost:5000

Step 3 — Start the frontend

bashcd frontend
cp .env.example .env.local

envVITE_API_URL=http://localhost:5000/api

bashnpm install
npm run dev

✅ Frontend: http://localhost:5173
