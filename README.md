✨ Overview

Café 3D is a production-grade, full-stack restaurant ordering and management platform that simulates a real-world food delivery and café management system.

It includes:

☕ Immersive 3D café experience powered by React Three Fiber
🍽️ Menu browsing, filtering, and food ordering workflows
💳 Secure checkout and payment integration
📦 Real-time order tracking using Socket.io
📅 Table reservation and booking management
🔐 JWT + Google OAuth authentication
📊 Role-based dashboards for customers, staff, and administrators

This project demonstrates production-level frontend and backend integration, scalable architecture, real-time communication, and modern full-stack engineering practices.

⚙️ Tech Stack
Layer	Technology
Frontend	React 18 + Vite + TypeScript
Styling	Tailwind CSS
State Management	Redux Toolkit + TanStack Query
Routing	React Router v6
3D Rendering	React Three Fiber + Drei
Animations	Framer Motion + GSAP
Forms	React Hook Form + Zod
Backend	Node.js + Express.js
Database	MongoDB + Mongoose
Authentication	JWT + Google OAuth
Real-Time	Socket.io
Payments	Stripe
Cloud Storage	Cloudinary
Deployment	Docker, Vercel, Netlify
📡 API Endpoints
Method	Endpoint	Description

🛠 Getting Started
1️⃣ Install Dependencies
cd frontend
npm install

cd ../backend
npm install
2️⃣ Run Backend
cd backend

npm run seed
npm run dev

Backend:

http://localhost:5000
3️⃣ Run Frontend
cd frontend

npm install
npm run dev

Frontend:

http://localhost:5173
🐳 Docker
Development
docker-compose --profile dev up --build
Production
docker-compose up --build -d
